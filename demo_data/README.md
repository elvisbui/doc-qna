# Demo Data

Sample documents for testing the doc-qna system.

## Files

- **machine-learning-basics.md** — Overview of machine learning concepts, types, and common algorithms. Markdown format.
- **project-management-guide.txt** — Best practices for project management. Plain text format.

## Usage

1. Start the doc-qna application (see the root README for instructions).
2. Open the Documents page in your browser.
3. Upload one or both files using the file upload interface.
4. Wait for processing to complete (status changes to "ready").
5. Switch to the Chat page and ask questions about the uploaded documents.

## Example Questions

After uploading the machine learning document:
- "What is the difference between supervised and unsupervised learning?"
- "Which algorithm is best for text classification?"
- "What is overfitting?"

After uploading the project management document:
- "What is the SMART framework?"
- "How should I handle changing requirements?"
- "What metrics should I track for project health?"
